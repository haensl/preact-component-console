# preact-component-console
Console emulator preact component.

## Quick Start

1. Install the package

    via NPM

    ```bash
    npm i --save preact-component-console
    ```

    via yarn
    ```bash
    yarn add preact-component-console
    ```

2. Use the component

```javascript
import Console from 'preact-component-console';

// ...

  render() {
    <Console />
  }
```
